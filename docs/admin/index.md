# Admin Panel Documentation

The Admin Panel is the primary interface for managing your institution. Below are the detailed guides for each module:

## 📋 Core Modules
- **[Student Management](student_management.md)**: Admission, profiles, houses, and alumni.
- **[Academics Management](academics.md)**: Classes, sections, subjects, and timetables.
- **[Fees Collection](fees_collection.md)**: Structuring fees, collection, and discounts.

## ⚙️ System Setup
- **System Settings**: General school configuration.
- **Roles & Permissions**: Managing staff access levels.

## 📊 Reports
- **Attendance Reports**: Tracking student and staff presence.
- **Financial Reports**: Income, expenses, and fee collection summaries.
