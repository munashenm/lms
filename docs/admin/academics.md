# Academic Management

The Academics module allows you to structure your school's educational framework.

## 🏫 Classes & Sections

The foundation of the LMS is the Class and Section structure.
- **Sections**: Create sections first (e.g., A, B, C).
- **Classes**: Create classes and assign one or more sections to them.
- Path: **Academics > Class / Section**

---

## 📚 Subjects & Subject Groups

Define the curriculum for your school.
- **Subjects**: Create individual subjects (e.g., Mathematics, Science, History). Mark them as Theory or Practical.
- **Subject Groups**: Group subjects for specific classes. For example, "Science Group" might include Physics, Chemistry, and Biology.
- Path: **Academics > Subjects / Subject Group**

---

## 📅 Class Timetable

Create a weekly schedule for each class and section.
1. Go to **Academics > Class Timetable**.
2. Select Class and Section.
3. Add time slots, subjects, and assigned teachers for each day of the week.

---

## 📖 Lesson Plan & Syllabus

Teachers can manage their teaching progress:
- **Syllabus Status**: Track which topics have been completed.
- **Lesson Plan**: Create daily or weekly teaching plans.
- Path: **Academics > Lesson Plan**

---

## 🏠 Assign Class Teacher

Assign a primary staff member to be responsible for a specific class and section. This gives them access to student data and attendance for that class.
- Path: **Academics > Assign Class Teacher**
