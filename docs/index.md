# Smart School LMS - User Manual & Documentation

Welcome to the official user manual for the **Smart School Learning Management System**. This documentation provides comprehensive guidance on how to set up, manage, and use the various modules of the LMS.

## 📊 Modules Overview

### 1. [Admin Panel](admin/index.md)
The central hub for school administrators to manage all aspects of the institution.
- **Student Information**: Admissions, Details, House, etc.
- **Academics**: Classes, Sections, Subjects, Timetable.
- **Finance**: Fees Collection, Income, Expenses.
- **Examinations**: Online Exams, Results, Admit Cards.
- **Human Resources**: Staff Directory, Payroll, Attendance.

### 2. [Staff & Teacher Panel](staff/index.md)
Tools for teachers to manage their classrooms and students.
- **Attendance**: Marking student attendance.
- **Homework**: Assigning and evaluating homework.
- **Syllabus**: Uploading study materials and lesson plans.

### 3. [Student & Parent Portal](portal/index.md)
A self-service portal for students and their parents.
- **My Profile**: View student details.
- **Fees**: Pay fees online and view receipts.
- **Progress**: Check attendance and examination results.
